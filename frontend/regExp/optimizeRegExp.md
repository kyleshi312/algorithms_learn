**正则运行阶段**

* 编译
* 设定起始位置
* 尝试匹配
* 匹配失败，下一位置继续第三步

**优化**

* 使用具体型字符组来代替通配符，来消除回溯
```
  /".*"/ // 校验123"abc"456中abc 会产生四次回溯
  /".*?"/  // 产生2次回溯
  /"[^"]*"/ // 完美
```


* 使用非捕获型分组
```
  /^[-]?(\d\.\d+|\d+|\.\d+)$/ 
  // 可以修改成：
  /^[-]?(?:\d\.\d+|\d+|\.\d+)$/
```

*  独立出确定字符
```
  /a+/ // 可以修改成 
  /aa*/
```
